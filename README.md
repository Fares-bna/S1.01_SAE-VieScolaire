# S1.01 SAE - Vie Scolaire
 
Projet de la SAÉ 1.01 (BUT Informatique) développé en C sous Visual Studio, en collaboration avec Ismail Fourra.
 
## Lancer le projet
- Ouvrir le fichier `S1.01.sln` avec Visual Studio
- Compiler puis exécuter le projet (Ctrl + F5)
 
## Explication du projet
Pour cette première initiation au développement en autonomie, le but de ce projet était de récréer un système d'administration de la vie scolaire d'un établissement en implémentant différentes fonctions et différentes gestion de variables de manière à être conforme aux exigences du sujet, on note par exemple la fonction inscription qui inscrit l'étudiant dont le nom est écrit derrière. 
Bien que d'un point de vu de dévelopement orienté objet cela semble assez simple, en langage C, l'approche est totalement différente et peut paraître complexe, surtout pour des nouveaux étudiants. 
Ce projet à donc permis de nous initier au développement en autonomie quelques semaines seulement après le début de l'année.
